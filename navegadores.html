Alguns Navegadores associados ao Desenvolvimento Web
Firefox Mozzila Firefox é um navegador livre e multo-plataforma desenvolvido pela Mozilla Foundation com a ajuda de centenas de colaboradores. A intenção da fundação é desenvolver um navegador leve, seguro, intuitivo e altamente extensível.
Chrome o Google Chrome é um navegador de internet, desenvolvido pela compahia Google com visual minimalista.
Internet Explorer Internet Explorer é uma série de navegadores web gráficos desenvolvidos pela Microsoft e inclusos como parte da linha de sistemas operacionais Microsoft Windows, iniciado em 1995.
Safari O Safari é um navegador desenvolvido pela Aplle Inc. e incluído como o navegador padrão a partir do sistema operacional Mac OS C v10.3 (Panther). Apresenta uma interface simples, característica dos produtos da Aplle.
Netscape Netscape Navigator é um navegador web proprietário que foi o principal produto da Netscape Comunication Corporation até 2002.
